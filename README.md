# Karddio Service API
Express JS service api used for the cardio-app capstone project, written on javascript ES6.

### Installation
1. Create .env file from .env.example, change everything you need
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run
   ```sh
   npm start
   ```
   or :
   ```sh
   node index.js
   ```
